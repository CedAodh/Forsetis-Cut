# FE5 Map Editor
This is a remnant of an editor I made around 2012.

# Execution environment
I think it requires Java 1.6 or 1.7.

# Use
1. Run FE5Editor.jar and open the ROM image.
2. Select the appropriate chapter, edit the map, and save it.

# Command help
Use the right mouse button to copy a tile, or hold to copy an entire range.

# Map Editor icon help
From left to right

- Ice: Compresses the current map and makes it writable.
- Pencil: Writes the compressed map to the ROM image (after writing, you need to save it with the purple floppy above)

- Left: Undo one operation
- Right: Redo one operation

- Chessboard: Select map tiles
- Colorwheel: Palette
- Square: Change map size

- <<: Previous map
- >>: Next map

# FE5 Map Editor
2012 年頃に作っていたエディタの残骸です。

# 実行環境
Java 1.6 か 1.7 あたりが必要だったと思います。

# 使い方
1. FE5Editor.jar を実行し、ROM イメージを開きます。
2. 適当に章を選び、マップを編集してセーブします。

# 操作補足
右クリックを押しながらの操作で範囲コピー

# Map Editor のアイコンの補足
左から順に

- 氷: 現在のマップを圧縮し、書き込みできる状態にします
- 筆: 圧縮したマップを ROM イメージに書き込み (書き込み後、上の紫のフロッピーで保存する必要あり)

- 左: 1 操作戻す
- 右: 1 操作やり直す

- 板: マップチップの選択
- 色: パレット
- □: マップサイズの変更

- <<: 1 マップ戻る
- >>: 1 マップ進む
